Accessibility Plugin JS

A plugin to create accessibility buttons

Quick start

First of all, put the code bellow inside the <head> of your website <link href="https://cdn.jsdelivr.net/gh/gustavocoimbradev/accessibility-plugin-js@main/dist/css/style.css" rel="stylesheet"> And then, put the code bellow inside and at the end of the <body> of your website <script src="https://cdn.jsdelivr.net/gh/gustavocoimbradev/accessibility-plugin-js@main/dist/js/script.js"></script>

Basic usage

This code will create a box with three options of accessibility. Feel free to customize it, but keep the classes .js-accessibility, .js-increase and .js-contrast. <div class="js-accessibility accessibility">
    <button class="js-increase increase">A+</button>
    <button class="js-decrease decrease">a-</button>
    <button class="js-contrast contrast">contrast</button>
</div>
