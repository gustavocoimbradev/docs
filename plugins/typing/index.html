Typing Plugin JS

A plugin to create a typewriter effect

Quick start

First of all, put the code bellow inside the <head> of your website <link href="https://cdn.jsdelivr.net/gh/gustavocoimbradev/typing-plugin-js@master/dist/css/style.css" rel="stylesheet"> And then, put the code bellow inside and at the end of the <body> of your website <script src="https://cdn.jsdelivr.net/gh/gustavocoimbradev/typing-plugin-js@master/dist/js/script.js"></script>

Basic usage

This code will create a box with a text with the typewriter effect. Feel free to customize it, but keep the class .typing-js <div class="typing-js">
    <span typing-text="Hello, I'm typing this text"></span>
</div>
